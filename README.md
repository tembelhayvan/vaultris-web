# Vaultris

A minimalist grid skipping puzzle game.

🌐 **Landing page:** https://tembelhayvan.github.io/vaultris-web/

## What is Vaultris?

A quiet puzzle game where you skip across a grid, trying to open every cell
without getting stuck. Simple rules, surprisingly deep.

Made by Aysegul, solo developer at [Garaj Projeleri](https://garajprojeleri.neocities.org).

## Game features

- **100 handcrafted levels** across 5×5 and 10×10 grids
- **4 game modes** — Normal, Hardcore, Time Attack, Target
- **Daily Challenge** with Wordle-style shareable results
- **25+ achievements** to unlock
- **Color-blind friendly** palette built in
- **Fully offline** — no ads, no tracking, no accounts
- Available in **English and Turkish**
